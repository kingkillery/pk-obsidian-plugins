# Memory Consolidation Workflow

## Session 1

- ask NotebookLM for structure
- ask for misconceptions
- create retrieval prompts

## Session 2

- answer prompts from memory
- reopen sources only after recall
- patch weak spots

## Session 3

- teach the idea in writing
- compress into a one-page guide
- generate one fresh example yourself

## Rule

Each pass should reduce dependence on the source and increase independence of explanation.
