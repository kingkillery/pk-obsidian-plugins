# Source Triage Checklist

Use this before adding material to NotebookLM.

- Does this source answer my current question?
- Is it primary, secondary, or commentary?
- Does it add a new angle, or repeat what I already have?
- Does it contain examples I can reuse for recall?
- Is it trustworthy enough for the claim I care about?

## Keep

- concise, high-signal sources
- sources that disagree in useful ways
- worked examples

## Drop

- redundant summaries
- broad motivational content
- source dumps without a clear learning goal
